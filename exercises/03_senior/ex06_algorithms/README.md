# Exercise 06: Algorithms

## Learning Objectives

- Implement classic algorithms
- Understand time and space complexity
- Apply recursion and dynamic programming

## Instructions

Fix the algorithm implementations in `exercise.py` to make all tests pass.

## Algorithms Covered

### Binary Search
Search in O(log n) time in sorted arrays.

### Merge Sort
Divide and conquer sorting in O(n log n).

### BFS/DFS
Graph traversal algorithms.

### Dynamic Programming
Optimize recursive problems with memoization.

## Complexity Reference

| Algorithm | Time | Space |
|-----------|------|-------|
| Binary Search | O(log n) | O(1) |
| Merge Sort | O(n log n) | O(n) |
| BFS/DFS | O(V + E) | O(V) |
