# Senior Level Exercises

Welcome to the **Senior** level! These exercises cover advanced Python concepts.

## Prerequisites

- Completed Junior and Mid level exercises
- Strong understanding of OOP and functional programming

## Exercises

| # | Exercise | Topic | Description |
|---|----------|-------|-------------|
| 01 | `ex01_async` | Async/Await | Asynchronous programming with asyncio |
| 02 | `ex02_metaclasses` | Metaclasses | Class creation and customization |
| 03 | `ex03_descriptors` | Descriptors | Property-like attributes with full control |
| 04 | `ex04_protocols` | Protocols & Types | Structural subtyping and type hints |
| 05 | `ex05_patterns` | Design Patterns | Common software design patterns |
| 06 | `ex06_algorithms` | Algorithms | Classic algorithm implementations |

## How to Start

1. Complete Junior and Mid levels first
2. Read documentation on advanced topics as needed
3. These exercises are challenging - take your time!

## Running Senior Exercises

```bash
# Run all Senior exercises
uv run pytest exercises/03_senior/ -v

# Run specific exercise
uv run pytest exercises/03_senior/ex01_async/ -v
```

## Success Criteria

Complete all 6 exercises (all tests passing) to achieve **Senior** mastery!

You've got this! 🏆
