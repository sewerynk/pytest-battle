# Mid Level Exercises

Welcome to the **Mid** level! These exercises cover intermediate Python concepts.

## Prerequisites

- Completed Junior level exercises
- Understanding of Python basics

## Exercises

| # | Exercise | Topic | Description |
|---|----------|-------|-------------|
| 01 | `ex01_comprehensions` | Comprehensions | List, dict, and set comprehensions |
| 02 | `ex02_classes` | OOP Basics | Classes, inheritance, and methods |
| 03 | `ex03_decorators` | Decorators | Function decorators and wrappers |
| 04 | `ex04_generators` | Generators | Generator functions and expressions |
| 05 | `ex05_exceptions` | Error Handling | Try/except, custom exceptions |
| 06 | `ex06_context_managers` | Context Managers | with statement and custom managers |

## How to Start

1. Complete Junior level first (recommended)
2. Begin with `ex01_comprehensions`
3. Work through sequentially - concepts build on each other

## Running Mid Exercises

```bash
# Run all Mid exercises
uv run pytest exercises/02_mid/ -v

# Run specific exercise
uv run pytest exercises/02_mid/ex01_comprehensions/ -v
```

## Success Criteria

Complete all 6 exercises (all tests passing) to advance to **Senior** level.

Keep pushing! 🚀
