# Junior Level Exercises

Welcome to the **Junior** level! These exercises cover Python fundamentals.

## Prerequisites

- Basic understanding of programming concepts
- Python 3.10+ installed

## Exercises

| # | Exercise | Topic | Description |
|---|----------|-------|-------------|
| 01 | `ex01_variables` | Variables & Types | Learn about Python's basic data types |
| 02 | `ex02_strings` | Strings | String manipulation and formatting |
| 03 | `ex03_lists` | Lists | Working with Python lists |
| 04 | `ex04_dictionaries` | Dictionaries | Key-value data structures |
| 05 | `ex05_functions` | Functions | Defining and calling functions |
| 06 | `ex06_control_flow` | Control Flow | Loops and conditionals |

## How to Start

1. Begin with `ex01_variables` and work your way through sequentially
2. Each exercise builds on concepts from previous ones
3. Read the `README.md` in each exercise folder
4. Fix the code in `exercise.py` to make tests pass
5. Check your progress with `uv run battle verify`

## Running Junior Exercises

```bash
# Run all Junior exercises
uv run pytest exercises/01_junior/ -v

# Run specific exercise
uv run pytest exercises/01_junior/ex01_variables/ -v
```

## Success Criteria

Complete all 6 exercises (all tests passing) to advance to **Mid** level.

Good luck! 🐍
