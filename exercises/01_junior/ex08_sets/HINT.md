# Hints for Exercise 08: Sets

## create_set
Use curly braces: `{1, 2, 3}`.

## remove_duplicates
Convert to set and back: `list(set(lst))`.

## union_sets
Use the `|` operator or `.union()` method: `a | b`.

## intersection_sets
Use the `&` operator or `.intersection()` method: `a & b`.

## difference_sets
Use the `-` operator or `.difference()` method: `a - b`.

## is_subset
Use the `.issubset()` method: `a.issubset(b)`.
Or use the `<=` operator: `a <= b`.
