# Hints for Exercise 01

## Hint 1: Returning Values

In Python, use the `return` keyword to return a value from a function:

```python
def get_number():
    return 5
```

## Hint 2: Type Conversion

Use built-in functions to convert between types:
- `int("42")` converts string to integer
- `str(42)` converts integer to string
- `float("3.14")` converts string to float

## Hint 3: Tuples

Create a tuple using parentheses:

```python
my_tuple = (1, 2, 3)
```

To swap values:

```python
def swap(a, b):
    return (b, a)
```
