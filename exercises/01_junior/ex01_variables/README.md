## Learning Objectives

- Understand Python's basic data types: `int`, `float`, `str`, `bool`
- Learn how to assign values to variables
- Understand type conversion

## Python Data Types

```python
# Integer
age = 25

# Float
price = 19.99

# String
name = "Alice"

# Boolean
is_valid = True
```

## Type Conversion

```python
# String to int
num = int("42")

# Int to string
text = str(42)

# String to float
decimal = float("3.14")
```
